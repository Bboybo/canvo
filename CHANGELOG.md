# Changelog

All notable changes to Canvo will be documented here.

## [0.9.0-beta.6] — 2026-03-05

### 🎉 Initial Public Beta Release

- **Canvas Engine** — Interactive D3 charts, tables, forms, quizzes rendered directly in chat
- **Linux Sandbox** — On-device terminal with 300+ commands, Python, Node.js support
- **Tool System** — Web search, calendar, contacts, clipboard, file I/O, TTS, notifications
- **Heartbeat** — Scheduled autonomous agent tasks
- **Memory** — Persistent knowledge base
- **Multi-Provider** — Google Gemini (with Google OAuth), Anthropic, OpenAI, Groq, Mistral, OpenRouter, Ollama, LM Studio
- **Google OAuth** — Sign in with Google for Gemini access (no API key needed)
- **Privacy-First** — All data on-device, BYOK model, no cloud sync
